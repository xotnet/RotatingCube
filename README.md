# RotatingCube
[Additional info]
OpenGL cube rotation with keyboard control support
__* tested on C++ 13__

![Alt Text](anim.gif)

[App support keyboard control]

**Space - pause/continue cube rotation**

**E - make cube bigger**

**Q - make cube smaller**

[Compilation]

**Using g++:** ```g++ main.cpp -lglfw3 -lopengl32 -lgdi32```
__* use lib/ libs for compile__
